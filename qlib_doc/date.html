日期¶

class date(const unsigned short yyyy, const unsigned short mm, const unsigned short dd)¶
Parameters:
yyyy: unsigned short
代表年份的四位数字。
mm: unsigned short
代表1-12个月的数字。
dd: unsigned short
代表一个月中的某一日的数字。

Date.year()¶

代表年份的数字。

Date.month()¶
Returns: int

Date.day()¶

Date.is_weekend()¶
Returns: bool

与 Date 相关的操作¶

Date 支持与整数的+， -， +=， -=¶

#include "qlib/date.h"

int main()
{
    auto d = Date(2022, 6, 16);
    std::cout << d << std::endl;
    std::cout << d + 1 << std::endl;
    std::cout << d - 1 << std::endl;

    d += 5;
    std::cout << d << std::endl;

    d -= 10;
    std::cout << d << std::endl;
}

//output:
    2022-06-16
    2022-06-17
    2022-06-15
    2022-06-21
    2022-06-11

Date 支持比较运算 >, >=, <, <=, ==, !=¶

#include "qlib/date.h"

int main()
{
    auto d1 = Date(2022, 6, 16);
    auto d2 = Date(2022, 6, 17);

    auto flag = d1 < d2;
    std::cout << flag << std::endl;
    // 1: True, 0: False

}

//output:
    1

Date 支持与 Period 及其子类（ days, weeks, months）的+-运算¶

See Period 子类.

备注

+-月份时， 遵循月末到月末保持日不变的原则

#include "qlib/date.h"
#include "qlib/period.h"

int main()
{
    auto d1 = Date(2022, 2, 28);
    auto p = Months(1);

    std::cout << d1 + p << std::endl;
    std::cout << d1 - p << std::endl;

}

//output:
    2022-03-31
    2022-01-31
