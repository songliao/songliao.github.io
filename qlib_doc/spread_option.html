差期权¶

差期权衡量两个资产价格之差作为标的价格的一种衍生品。

期权名称

收益描述

差期权
看涨 \(\text{Max}(S_1 - S_2 - X, 0)\)
看跌 \(\text{Max}(X - S_1 + S_2, 0)\)
Pv(double spot1, double spot2, double rho, double r, double b1, double b2, double vol1, double vol2, double ttm, OptionType type, double strike， double quantity1, double quantity2);
Parameters:
spot1: double
资产1的价格。
spot2: double
资产2的价格。
rho: double
两个挂钩标的之间的相关性。
r: double
无风险利率。
b1: double
资产1的持仓成本。
b2: double
资产2的持仓成本。
vol1: double
资产1的波动率。
vol2: double
资产2的波动率。
ttm: double
到期时间。
type: OptionType
期权类型, CALL 或 PUT。
strike: double
行权价格。
quantity1: double
资产1的数量。
quantity2: double
资产2的数量。
Returns: double
返回差期权的现值。

Example¶

#include <gtest/gtest.h>
#include <iostream>

#include "qlib/analytical_engine/spread_option.h"

TEST(TestAnalytical, SpreadOption)
{
    // page 174
    double spot1 = 122;
    double spot2 = 120;
    double rho = -0.5;
    double r = 0.1;
    double b1 = 0.0;
    double b2 = 0.0;
    double vol1 = 0.2;
    double vol2 = 0.2;
    double t = 0.1;
    OptionType type = OptionType::CALL;
    double strike = 3;
    double quantity1 = 1;
    double quantity2 = 1;

    double pv =  AnalyticalEngine::Spread::Pv(spot1, spot2, rho, r, b1, b2, vol1, vol2, t, type, strike, quantity1, quantity2);

    EXPECT_NEAR(4.7530, pv, 1e-4);
}
