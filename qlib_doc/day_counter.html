Day Counter¶

class DayCounter¶

Day Counter 是用来计算给定两个日期之间距离的一个特定的市场惯例规则，包括按天或者年化计数。

DayCounter 的继类¶

Act365¶

class Act365¶

Act365 表示实际天数/365天的市场惯例规则。

Bus244¶

class Bus244(calendar)¶

Bus244 表示交易日/244天的市场惯例规则。

Parameters:
calendar: Calendar
一个日历对象。

备注

当使用 Bus244 市场惯例规则时，日历应当指定一个给定的txt节假日列表或者一组节假日假期。
