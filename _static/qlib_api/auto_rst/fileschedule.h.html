生成交易日¶

Enums

enum class BusinessDayConvention¶

函数在生成交易日时，如遇非交易日的调整规则。

Values:

enumerator FOLLOWING¶

调整为下一交易日。

enumerator MODIFIED_FOLLOWING¶

若下一交易日和非交易日同月，则调整为下一交易日，否则调整为上一交易日。

enumerator PRECEDING¶

调整为上一交易日。

enumerator MODIFIED_PRECEDING¶

若上一交易日和非交易日同月，则调整为上一交易日，否则调整为下一交易日。

enumerator HALF_MONTH_MODIFIED_FOLLOWING¶

若下一交易日和非交易日在同半月（按每月15号划分），则调整为下一交易日，否则调整为上一交易日。

enumerator NEAREST¶

调整为最近的交易日。

enum class DateGenerationRule¶

date generation rules

Values:

enumerator FORWARD¶

从给定的交易日向前生成。

enumerator BACKWARD¶

从给定的交易日向后生成。
