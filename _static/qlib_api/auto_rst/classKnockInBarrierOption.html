class KnockInBarrierOption¶

单障碍敲入型期权。

Public Functions

inline KnockInBarrierOption(BarrierType b_type, std::vector<Date> &observation_dates, std::vector<double> &barriers, Payoff &ki_payoff, Payoff &nki_payoff, Calendar &cal)¶
参数

  • b_type – 见BarrierType

  • observation_dates – 一个 vector<Date> ，给出敲入观察日

  • barriers – 一个 vector<double> ，与 observation_dates 一一对应

  • ki_payoff – 一个 Payoff ，如果发生敲入，则在期末生效

  • nki_payoff – 一个 Payoff ，如果不发生敲入，则在期末生效

  • cal – Calender ，日历规则
