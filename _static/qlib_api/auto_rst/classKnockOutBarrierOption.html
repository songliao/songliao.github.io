class KnockOutBarrierOption¶

单障碍敲出型期权。

Public Functions

inline KnockOutBarrierOption(BarrierType b_type, std::vector<Date> &observation_dates, std::vector<double> &barriers, std::vector<Payoff> &knock_payoffs, Payoff &maturity_payoff, Calendar &cal)¶
参数

  • b_type – 见BarrierType

  • observation_dates – 一个 vector<Date>，给出敲出观察日

  • barriers – 一个 vector<double>，与 observation_dates 一一对应

  • knock_payoffs – vector<Payoff>，如果发生敲出，则在敲出日生效

  • maturity_payoff – 一个 Payoff ，如果不发生敲出，则在期末生效

  • cal – Calendar ，日历规则
