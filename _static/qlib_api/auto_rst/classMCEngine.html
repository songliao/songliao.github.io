蒙特卡洛定价引擎MCEngine¶

class MCEngine¶

MCEngine 中包含了蒙特卡洛模拟的参数，包括模拟次数、随机数种子等。整个蒙特卡洛模拟 将被等分成几个次数较少的模拟，以提升性能。使用方法如下：

Public Functions

inline explicit MCEngine(unsigned int n_sim, unsigned int batch_size = 10000, bool parallel = true, unsigned int seed = std::time(nullptr))¶
参数

  • n_sim – 模拟的路径总数

  • batch_size – 几个次数较少的模拟的路径数，默认为10000

  • parallel – 是否使用OpenMP进行并行计算

  • seed – 随机数种子
